# News Hub - 信息汇总中心

## 产品需求文档 (PRD)

### 项目概述

**项目名称：** News Hub (信息汇总中心)  
**创建日期：** 2026-02-28  
**负责人：** 暖煦sunny  
**开发者：** 阿瓜 (AI Assistant)

### 产品目标

为用户提供一个高质量的财经和科技信息汇总平台，每日精选重要资讯，帮助用户快速了解市场动态。

### 核心功能

1. **每日摘要**
   - 财经摘要
   - 科技摘要
   - 可展开查看详情

2. **新闻展示**
   - 分类展示（财经/科技）
   - 热门话题排行
   - 点击跳转原文

3. **多语言支持**
   - 中文/英文切换
   - 完整翻译支持

4. **信息来源**
   - 中文财经媒体
   - 英文财经媒体
   - 科技媒体
   - 数据工具

### 目标用户

- 金融从业者
- 科技爱好者
- 投资者
- 关注财经科技动态的用户

### 技术方案

- **前端：** 纯 HTML/CSS/JavaScript
- **部署：** Cloudflare Tunnel
- **数据源：** Tavily Search API
- **AI处理：** GLM-5 模型
